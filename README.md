## LoucoUtils
A chattrigger script for Hypixel Skyblock
## About
Personal project made for own use using mod [Chat Triggers} (https://chattriggers.com) for Minecraft Server Hypixel
As Hypixel's recent updates to force more recent versions, this project is abandoned 

## What does it do?
### Fishing Features
- Sea Creatures Alerts
### Dungeon Features
- Auto Requeue
- Auto join after kick
- Falling Blocks Disabler
### General Features
- Vanquisher Alert
- Golem Particles Disabler
- Waypoints

## General Commands
- /lu or /loucoutils for general acess to all features
- /f1, /f2, /f3, etc to join regular dungeon floors
- /m1, /m2, /m3, etc to join master mode dungeon floors
- /aw or /autowarp to toggle the Auto Warp feature
- /sendlimbo or /goafk to bruteforce player into limbo
